JavaScript Toolkit Application & Product

Application Frameworks

Frameworks provides you with an opinionated structure that helps you build your application faster.

AngularJS

AngularJS is a framework that ships with two-way databind, XHR utilities, model and components for creating interactive webpages

Links

  • https://angularjs.org/

Backbone.js

Backbone.js provides structure conventions for web applications, with models, views and RESTful integration, that allows you to integrate with other libraries to provide other functionalities. There are several other frameworks built on top of Backbone.js there comes with more features.

Links

  • http://backbonejs.org/

Component JS

A library to build interactive components and guide the structuring of user interface elements.

Links

  • http://componentjs.com/

Ember

Ember is a framework to build interactive web apps with configuration as convention to improve productivity.

Links

  • http://emberjs.com

Google Closure

Google Closure is a cross-browser library for network, DOM manipulation, animation and datastructures.

Links

  • https://developers.google.com/closure/library/

Knockout

A library that provides models with two-way databindings for the DOM.

Links

  • http://knockoutjs.com/

React

A library to build your views, backed by a virtual DOM to speed up the interaction with the rendering engines.

Links

  • https://facebook.github.io/react/
